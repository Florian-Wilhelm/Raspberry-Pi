# (IoT) Geiger counter

## Description

Radiation measurement device with Raspberry Pi Pico and STS-5 tube.

## Test Software 

V3.93 -> OLED display, System clock <3 MHz

V4.18 -> OLED display, System clock 12 MHz, IoT / BLE with RN4870 IC
